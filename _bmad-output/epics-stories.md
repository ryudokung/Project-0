# Epics & User Stories - Project-0

## Epic 1: User Identity & Onboarding
**Goal:** Allow users to securely enter the game using their social accounts or Web3 wallets.

- **Story 1.1 (Web3 Auth):** As a player, I want to login via X or Google so that I can start playing without managing private keys.
- **Story 1.2 (Embedded Wallet):** As a new player, I want an automatic embedded wallet so that I can interact with the blockchain seamlessly.
- **Story 1.3 (User Profile):** As a player, I want to see my credits and basic stats on a dashboard.

## Epic 2: The Engineering Bay (Assembly)
**Goal:** Implement the core AI-driven NFT creation process using the Saga Pattern.

- **Story 2.1 (Initiate Assembly):** As a player, I want to spend resources/USDT to start building a Mech/Ship.
- **Story 2.2 (AI Generation):** As a system, I want to generate unique visuals using FLUX.1 based on the vehicle type and class.
- **Story 2.3 (HITL Approval):** As an admin, I want to approve AI-generated images before they are minted as NFTs.
- **Story 2.4 (On-chain Minting):** As a player, I want my approved Mech to be minted on Base L2 automatically.

## Epic 3: Deep Space Exploration
**Goal:** Create the multi-stage exploration loop.

- **Story 3.1 (Space Travel):** As a player, I want to use my Ship to travel to distant stars.
- **Story 3.2 (Planetary Surface):** As a player, I want to deploy my Mech to explore the surface of a discovered planet.
- **Story 3.3 (Star Discovery):** As a community, we want new stars to be discovered based on collective exploration progress.

## Epic 4: Tactical Combat
**Goal:** Implement the stat-based battle engine with AI visualizations.

- **Story 4.1 (Battle Engine):** As a player, I want to engage in combat where my Mech's stats determine the outcome.
- **Story 4.2 (Action Shots):** As a player, I want to see an AI-generated image of the battle's climax.
- **Story 4.3 (Battle Logs):** As a player, I want to view my past battle results for 7 days.

## Epic 5: Galactic Economy
**Goal:** Manage monetization and difficulty adjustment.

- **Story 5.1 (Season Pass):** As a player, I want to buy a Season Pass to get exploration buffs.
- **Story 5.2 (Difficulty Adjustment):** As a system, I want to adjust resource drop rates based on total player activity (Bitcoin-style).
- **Story 5.3 (Marketplace):** As a player, I want to trade my NFT "Keys" with other players for USDT.
